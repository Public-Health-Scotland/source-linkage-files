# Read Acute extract

Read Acute extract

## Usage

``` r
read_extract_acute(
  year,
  file_path = get_boxi_extract_path(year = year, type = "acute")
)
```

## Arguments

- year:

  Financial year for the BOXI extract.

- file_path:

  BOXI extract location

## Value

a [tibble](https://tibble.tidyverse.org/reference/tibble-package.html).
