# Add SDS columns

Add SDS columns

## Usage

``` r
add_sds_columns(episode_file, prefix, condition)
```

## Arguments

- episode_file:

  Tibble containing episodic data.

- prefix:

  Prefix to add to related columns, e.g. "Acute"

- condition:

  Condition to create new columns based on
