# Get the SLF individual file path

Get the SLF individual file path

## Usage

``` r
get_slf_individual_path(year, ...)
```

## Arguments

- year:

  Financial year

- ...:

  additional arguments passed to
  [`get_file_path()`](https://public-health-scotland.github.io/source-linkage-files/reference/get_file_path.md)

## Value

Path to the final individual file
