# Create LCA test flags

Create flags for Local Authorities

## Usage

``` r
create_lca_test_flags(data, lca_var)
```

## Arguments

- data:

  the data containing an LCA variable

- lca_var:

  LCA variable e.g. CA2019, CA2011

## Value

a dataframe with flag (1 or 0) for each LCA
