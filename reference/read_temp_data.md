# Read a temp data from disk for debugging purpose

Read a temp data to disk for debugging purpose.

## Usage

``` r
read_temp_data(year, file_name)
```

## Arguments

- year:

  year variable

- file_name:

  The file name to be read

## Value

the data for next step as a
[tibble](https://tibble.tidyverse.org/reference/tibble-package.html).
