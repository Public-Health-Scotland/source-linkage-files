# Create HSCP test flags

Create flags for Health & Social Care Partnerships

## Usage

``` r
create_hscp_test_flags(data, hscp_var)
```

## Arguments

- data:

  the data containing a HSCP variable

- hscp_var:

  HSCP variable e.g. HSCP2019 HSCP2018

## Value

a dataframe with flag (TRUE or FALSE) for each HSCP
