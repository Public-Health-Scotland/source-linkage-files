# Convert GP Practice Codes to a Dummy Code

Convert English GP practice codes to a dummy code

## Usage

``` r
convert_eng_gpprac_to_dummy(gpprac, dummy_code = 9995L)
```

## Arguments

- gpprac:

  A character vector containing the GP practice codes

- dummy_code:

  The dummy code to use. Default is 9995

## Value

An integer vector with only Scottish GP codes
