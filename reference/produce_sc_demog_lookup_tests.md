# Social Care Demographic Lookup Tests

Produce the tests for Social Care Demographic Lookup

## Usage

``` r
produce_sc_demog_lookup_tests(data)
```

## Arguments

- data:

  new or old data for testing summary flags (data is from
  [`get_sc_demog_lookup_path()`](https://public-health-scotland.github.io/source-linkage-files/reference/get_sc_demog_lookup_path.md))

## Value

a dataframe with a count of each flag.
