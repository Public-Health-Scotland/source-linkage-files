# Clean temp data from disk

Clean temp data from disk to save storage.

## Usage

``` r
clean_temp_data(year, file_type = c("ep", "ind"))
```

## Arguments

- year:

  year variable

- file_type:

  ep or ind files

## Value

the data for next step as a
[tibble](https://tibble.tidyverse.org/reference/tibble-package.html).
