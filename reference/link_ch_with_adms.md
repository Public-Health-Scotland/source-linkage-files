# Link AE episodes released from CH

1.  Add two variable, sc_ch_link_ae and sc_ch_link_adms, for CH episode
    to indicate discharge to AE2 and (01B, 04B, GLS) separately.

2.  Populate ch_name and ch_postcode to relevant AE episodes from CH
    episodes

## Usage

``` r
link_ch_with_adms(ep)
```

## Arguments

- ep:

  episode file
