# Social care Client lookup Tests

Produce the test for the social care Client all episodes

## Usage

``` r
produce_tests_sc_client_lookup(data)
```

## Arguments

- data:

  new or old data for testing summary flags (data is from
  [`get_source_extract_path()`](https://public-health-scotland.github.io/source-linkage-files/reference/get_source_extract_path.md))

## Value

a dataframe with a count of each flag.
