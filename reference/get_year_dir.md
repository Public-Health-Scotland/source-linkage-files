# Year Directory

Get the directory for Source Linkage File Updates for the given year.

## Usage

``` r
get_year_dir(year, extracts_dir = FALSE)
```

## Arguments

- year:

  The Financial Year e.g. 1718

- extracts_dir:

  (optional) Whether to return the Extracts folder (`TRUE`) or the
  top-level folder (`FALSE`).

## Value

The file path to the year directory (on sourcedev)
