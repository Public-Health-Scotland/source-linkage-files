# Fill any missing CIJ markers for records that should have them

Fill any missing CIJ markers for records that should have them

## Usage

``` r
fill_missing_cij_markers(data)
```

## Arguments

- data:

  The in-progress episode file data.

## Value

A data frame with CIJ markers filled in for those missing.
